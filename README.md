# Maxclicker Download

This repository hosts the Maxclicker download page through GitHub Pages and stores release assets through GitHub Releases.

## What to do

1. Keep `index.html` in the repository root.
2. Enable GitHub Pages from the `main` branch and `/ (root)` folder.
3. Create a Release with tag like `v1.0.1`.
4. Upload `AutoPotionModern.exe` as a release asset.
5. Use this page URL in the app remote config:
   - `https://crossgozon.github.io/maxclicker-download/`

## Direct release asset pattern

After a release is published, the direct asset URL will look like this:

`https://github.com/crossgozon/maxclicker-download/releases/download/v1.0.1/AutoPotionModern.exe`
